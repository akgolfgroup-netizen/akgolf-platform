## 📱 RESPONSIVE DESIGN PRINSIPPER (Gjelder alle prompts)

```
BREAKPOINTS:
- Mobile: < 640px
- Tablet: 640px - 1024px  
- Desktop: > 1024px

MOBILE ADAPTATIONS:
- Touch targets minimum 44px (preferably 56px)
- Simplified navigation (hamburger menu)
- Stacked layouts (single column)
- Horizontal scroll for card rows
- Swipe gestures for common actions
- Bottom navigation bar for main sections
- Floating Action Button (FAB) for primary action

TABLET ADAPTATIONS:
- 2-column grid layouts
- Sidebar navigation collapsible
- Touch + mouse interactions
- Medium-sized cards

DESKTOP ADAPTATIONS:
- Multi-column bento grids
- Hover states with detailed tooltips
- Drag-and-drop functionality
- Keyboard shortcuts visible
- Wide layouts with ample whitespace

ANIMATION REDUCTION:
- Respect prefers-reduced-motion
- Simplify animations on mobile for performance
- Essential animations only on low-power mode
```

## ⚡ GENERERINGSTIPS FOR STITCH

1. **Generer én skjerm om gangen**
2. **Start med Desktop** - Stitch skalerer ned automatisk
3. **Hvis resultat ikke er perfekt**: Bruk "Edit with Text" for justeringer
4. **Farger**: Nevn hex-koder eksplisitt i prompt
5. **For variasjoner**: Bruk "Generate Variants" etter første generering
6. **Eksport**: Hent ut som React-komponenter når fornøyd
