# **Client's Request**

"We're looking for a web app design for Homiva, focused on simplifying the home-buying process. The app should offer intuitive search, personalized property recommendations, and seamless user experience, helping people navigate real estate decisions with ease and confidence."

# **About Project**

Homiva is a real estate web app designed to simplify property searches and home-buying decisions. It offers personalized property recommendations, easy communication with agents, and streamlined search filters to help users find their ideal home effortlessly.

# **User Challenges**

Homiva faced challenges with user navigation and cluttered property listings, making it difficult for users to quickly find relevant homes. Additionally, communication between buyers and agents was fragmented, leading to missed opportunities and delays.

# **Valuable Insights**

Through user research, we discovered that 65% of Homiva users struggled with complex property filters, and 72% found agent communication tools lacking. By focusing on these pain points, we refined the app’s design to address these core user frustrations.

# **Project Result**

By simplifying Homiva’s property search filters and improving communication features, we reduced user search time by 40% and increased successful agent connections by 25%. The streamlined design resulted in a more efficient and user-friendly home-buying experience.

# **Client's Feedback**

“We're excited to collaborate on the next phase of Homiva, developing an adaptive version for tablets and mobile. This will allow users to seamlessly transition between devices, improving accessibility and enhancing their property search experience.”
