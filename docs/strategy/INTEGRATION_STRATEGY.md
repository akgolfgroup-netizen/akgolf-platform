# Integrasjonsstrategi: Treningsplanlegger + Analyse + DataGolf

---

## 🎯 VISJON

**"Gjør treningsplanlegging fra 'gjetting' til 'data-drevet vitenskap'"**

Når spilleren åpner treningsplanleggeren, skal de ikke bare se en tom kalender - de skal se en **personlig optimalisert treningsplan** basert på:
- Deres egne styrker og svakheter
- DataGolf Tour-benchmarks
- Hva som gir raskest fremgang
- Hvor mye tid de har tilgjengelig

---

## 📊 NÅVÆRENDE VS FREMTIDIG

### NÅVÆRENDE (I dag)
```
Spiller: "Jeg skal trene i morgen... hva skal jeg gjøre?"

Handling:
1. Dra inn "Driving range" template
2. Gjette seg frem til øvelser
3. Håpe det gir resultater

Resultat: 60% adherence, usikker fremgang
```

### FREMTIDIG (Med DataGolf)
```
AI COACH: "Basert på dine data anbefaler jeg:"

📊 ANALYSE:
• Din approach fra 100m: 28ft proximity (65% Tour Score)
• Tour median: 21ft proximity
• Gap: -0.5 SG per runde = 2.5 HCP-poeng!

🎯 ANBEFALING:
I morgen: 60 min approach-fokus
• 20 min: 100m Precision Challenge
• 20 min: Avstandskontroll 125-150m  
• 20 min: Test deg mot Viktor Hovland!

Forventet resultat: -0.1 SG på 4 uker
```

### 1. Smart Kalender (Treningsplanlegger)

#### Nå: Tom kalender → Fremtid: Fylt med forslag

```typescript
// Når kalenderen lastes:
async function loadSmartCalendar(userId: string, weekStart: Date) {
  
  // 1. Hent spillerens data
  const playerData = await getPlayerData(userId);
  
  // 2. Hent DataGolf-benchmarks
  const tourBenchmarks = await getTourBenchmarks();
  
  // 3. Analyser gaps
  const gaps = analyzeGaps(playerData, tourBenchmarks);
  
  // 4. Generer ukens anbefalinger
  const weeklyPlan = generateWeeklyPlan({
    gaps,
    availableHours: playerData.preferences.hoursPerWeek,
    facilities: playerData.preferences.facilities,
    previousAdherence: playerData.trainingHistory.adherenceRate
  });
  
  // 5. Returner kalender med:
  return {
    // Eksisterende planlagte økter
    plannedEvents: [...],
    
    // NYTT: AI-forslag til tomme slots
    suggestedEvents: weeklyPlan.sessions.map(session => ({
      type: 'SUGGESTION',
      title: session.focus,
      reason: session.reasoning, // "Din approach er 0.5 SG bak Tour"
      exercises: session.exercises,
      isAutoGenerated: true
    })),
    
    // NYTT: "Beat the Pro" utfordringer
    challenges: weeklyPlan.challenges
  };
}
```
