Security & Access

Authentication Instructions

AK Sports OS uses API keys to allow access to the API. You can register a new API key at our dashboard. Your API keys carry many privileges, so be sure to keep them secure!

Authorization: Bearer YOUR_SECRET_KEY
GET /v2/athletes/list

List all Athletes

Returns a paginated list of all active athletes within your organization. Use the limit and offset parameters to control the data returned.

Parameters
limit Integer

The number of objects to return. Max 100.

team_id String

Filter by specific team identifier.
