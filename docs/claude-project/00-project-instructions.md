## Viktige regler

### ALDRI
- Bruk emojier (bruk Lucide-ikoner)
- Nevn MVA
- Vis sertifiseringer (PGA, TrackMan Certified, etc.)
- Skriv "kortspill" (riktig: "narspill")
- Skriv "elev" (riktig: "spiller")
- Bruk bronse/gull farger (#B07D4F)
- Bruk buzzwords ("revolusjonerende", "transformere")

### ALLTID
- Norsk bokmal
- Direkte og konkret tone
- Priser i kroner (ikke ore)
- "Ingen binding" ved abonnement
- Svart (#1D1D1F) og hvit design
- CTA-gront (#16a34a) for handlingsknapper
