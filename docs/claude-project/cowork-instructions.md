## The Foundation Method

Vår coaching-filosofi med fem faser:

1. BUILD — Bygg grunnmuren (teknikk, grep, setup)
2. STAB — Stabiliser under press (repetisjon, mestring)
3. TEST — Test under stressor (variasjon, utfordring)
4. TRANSFER — Overfør til banen (fra range til spill)
5. PERFORM — Prestér i konkurranse (mental styrke, strategi)

Treningspyramiden: FYS → TEK → SLAG → SPILL → TURN

Prinsipper:
- Fundament først
- Systematisk progresjon
- Data erstatter følelser (TrackMan + spillerportal)
- Hele spilleren (teknikk + strategi + fysisk + mental)
- Individuell plan, felles standard
