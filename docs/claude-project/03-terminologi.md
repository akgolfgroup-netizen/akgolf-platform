## SLAG-kategorier (Strokes Gained-basert)

| Kategori | SG-andel | Treningsandel | Beskrivelse |
|----------|----------|---------------|-------------|
| Tee Total | 26% | 30-35% | Driver/tre fra tee |
| Approach | 17% | 25-30% | Innspill >100m |
| Narspill | 17% | 20-25% | Slag <100m (ikke putting) |
| Putting | 40% | 15-20% | Alle putts |

**Viktig:** Putting har storst SG-andel (40%), men lavest treningsandel fordi forbedring er vanskeligere a oppna.

## Treningspyramiden

Trening bygges nedenfra og opp:

```
        TURN
       /    \
     SPILL
    /      \
   SLAG
  /        \
 TEK
/            \
    FYS
```

| Nivå | Navn | Beskrivelse |
|------|------|-------------|
| 1 | **FYS** | Fysisk trening (gym, kondisjon, mobilitet) |
| 2 | **TEK** | Teknikk (bevegelsesmonster, posisjoner) |
| 3 | **SLAG** | Golfslag (slagkvalitet, avstand, noyaktighet) |
| 4 | **SPILL** | Spill (banemanagement, strategi) |
| 5 | **TURN** | Turnering (konkurranseforberedelse) |

**Prinsipp:** Du kan ikke bygge SLAG pa darlig TEK. Du kan ikke bygge TEK pa darlig FYS.

| Fase | Fokus | Beskrivelse |

| ARM | Armbevegelse | Armsvingen isolert |
| KOLLE | Kollebevegelse | Kollehodet, face control |
| BALL | Ballkontakt | Treff, ballflukt |
| AUTO | Automatisering | Ubevisst kompetanse |

## Periodisering

| Periode | Fokus | Nar |
|---------|-------|-----|
| **GRUNNPERIODE** | Bygg fundament (FYS + TEK) | Vinter/tidlig var |
| **SPESIALISERING** | Overfor TEK til SLAG | Var |
| **TURNERING** | Konkurranseforberedelse | Sommer |

## Decade Golf-termer

| Term | Betydning |
|------|-----------|
| Strokes Gained (SG) | Slag vunnet/tapt vs. referansespiller |
| SG: Tee | Prestasjon fra tee |
| SG: Approach | Prestasjon pa innspill |
| SG: Around the Green | Prestasjon i narspill |
| SG: Putting | Prestasjon pa green |
| Expected Strokes | Forventet antall slag fra gitt posisjon |
