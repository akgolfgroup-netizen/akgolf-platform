# AK Golf Platform — Dokumentasjon

Aktiv prosjektdokumentasjon. For utgåtte dokumenter, se `archive/`.

## Struktur

- `architecture/` — systemarkitektur, dataflyt, integrasjonsdiagrammer
- `api/` — API-kontrakter, endepunkter, webhook-spesifikasjoner
- `onboarding/` — onboarding for nye utviklere/AI-agenter
- `decisions/` — ADR (Architecture Decision Records)

## Andre dokumentkilder

- `WORKLOG.md` (rot) — arbeidslogg
- `CLAUDE.md` (rot) — AI-instrukser
- `.claude/rules/` — detaljerte prosjektregler
- `docs/status/` — prosjektstatus (BACKLOG, audits)
