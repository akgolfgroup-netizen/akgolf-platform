## 3. KONKRET JURIDISK ANALYSE

### Scenario 1: Vi bruker "5% Buffer"

```
SPØRSMÅL: Kan Scott Fawcett saksøke oss for å bruke "5% Buffer"?

SVAR: Sannsynligvis nei, men med forbehold.

ANALYSE:
• "5% Buffer" er et KONSEPT, ikke en konkret algoritme
• Matematikken bak (95% konfidensintervall) er standard statistikk
• Faguttrykk kan ikke varemerkeregistreres i seg selv

MEN:
• Hvis DECADE har PATENT på spesifikk implementasjon:
  "System for å beregne golf-strategi basert på 
   kontekst-avhengig spredning med 5% buffer..."
  → Da kan vi komme i trøbbel

RISIKO: 🟡 Medium
```

### Scenario 2: Vi implementerer "Shotgun Pattern"

```
SPØRSMÅL: Kan vi bruke begrepet "shotgun pattern"?

SVAR: Ja, sannsynligvis.

ANALYSE:
• "Shotgun vs rifle" er et generelt metafor
• Brukes i mange sammenhenger (skyting, business, etc)
• Ikke originalt for DECADE

MEN:
• Hvis vi kopierer EKSAKT formulering fra DECADE-materiale:
  "Din spredning er en shotgun, ikke en rifle shot"
  → Kan være copyright-krenkelse

ANBEFALING:
• Bruk egen formulering:
  "Din spredning er bred - sikte deretter"
  "Sprednings-basert strategi"
  "Kontekst-tilpasset siktepunkt"

RISIKO: 🟢 Lav
```

### Scenario 3: Vi kombinerer DECADE + Mental

```
SPØRSMÅL: Er dette en krenkelse?

SVAR: Nei, nesten sikkert ikke.

ANALYSE:
• Kombinasjon av to eksisterende konsepter er nytt
• Ingen har patent på "DECADE + mental trening"
• Dette er en FORBEDRING, ikke kopi

FORDEL FOR OSS:
• Dette kan VI patentere!
• "System for integrasjon av mental trening med 
   statistisk golf-strategi"

RISIKO: 🟢 Lav (faktisk en fordel)
```

### Scenario 4: Vi bruker "DECADE" navnet

```
SPØRSMÅL: Kan vi si "Basert på DECADE-prinsipper"?

SVAR: NEI! Dette er varemerke-krenkelse.

ANALYSE:
• "DECADE" er nesten sikkert registrert varemerke
• Å bruke navnet skaper forveksling
• Kan stanse salg og kreve erstatning

KONSEKVENSER:
• Rettssak
• In injunction (forbud mot å bruke navnet)
• Erstatningskrav
• Exit-verdi reduseres dramatisk

RISIKO: 🔴 Høy (ikke gjør dette!)
```

## 7. JURIDISKE STEG VI MÅ TA

### Før Lansering

```
1. VAREMERKE-SØK (UKE 1)
   □ Sjekk om "DECADE" er registrert i Norge
   □ Sjekk internasjonale registre (EUIPO, USPTO)
   □ Kostnad: ~5.000 kr via patentbyrå
   
2. VAREMERKE-REGISTRERING (UKE 2-4)
   □ Registrer "AK Golf", "Smart Caddy", "Strategi-System"
   □ Klasser: Software, utdanning, sport
   □ Kostnad: ~15.000 kr (Norge) + 25.000 kr (EU)

3. PATENT-UNDERSØKELSE (UKE 2-3)
   □ Søk i patentdatabaser (Espacenet, Google Patents)
   □ Søkeord: "golf strategy", "statistical golf", "dispersion"
   □ Kostnad: ~10.000 kr (profesjonell søk)
   
4. OPPLYSNING OM INSPIRASJON (FRIVILLIG)
   □ Vurder å kontakte Scott Fawcett
   □ Tilbud: "Vi har latt oss inspirere, men utviklet eget system"
   □ Tilbud: Revenue share (0.5-1%) for å unngå konflikt
```

### Ved Exit

```
5. DUE DILIGENCE (Før salg)
   □ Juridisk vurdering av IP-portefølje
   □ Bekreftelse på at vi ikke krenker DECADE
   □ "Clean IP" attestasjon
   
6. INDEMNITY (I salgsavtale)
   □ Selger (du) garanterer at IP er clean
   □ Kjøper har rett til erstatning hvis rettssak oppstår
   □ Forsikring for IP-krav (finnes som produkt)
```

## 9. ØKONOMISK KONFLIKT-ANALYSE

### Hva vil en rettssak koste?

```
SCENARIO: DECADE saksøker oss

KOSTNADER FOR OSS:
• Juridisk forsvar (Norge): 500.000 - 2.000.000 kr
• Juridisk forsvar (USA hvis relevant): 2-5M kr
• Tapt inntekt under sak: 500.000 kr/mnd
• Omdømme-tap: Umulig å kvantifisere
• Exit-verdi redusert: 30-50%

TOTALT: 5-15M kr + exit-tap

KOSTNADER FOR DECADE:
• Juridiske kostnader: 3-10M kr
• Tid og ressurser
• Risiko for å tape (hvis vi har god sak)

SANNSYNLIGHET FOR SØKSMÅL:
• Hvis vi er små (under 100 brukere): 5%
• Hvis vi vokser (over 1.000 brukere): 30%
• Hvis vi er tydelig kopi: 70%
• Hvis vi er inspirert men unike: 20%
```

### Break-Even for DECADE

```
DECADE vil bare saksøke hvis:
• De tror de kan vinne (sterk sak)
• Vi har penger å ta (saksøker ikke blodslanger)
• Det er strategisk viktig (hindre kopier)
• Kostnaden er mindre enn potensiell tap

VI ER TRYGGEST HVIS:
• Vi er tydelig unike (mental + strategi)
• Vi er store nok til å forsvare oss, men ikke så 
  store at vi er trussel
• Vi har juridisk ryddighet på plass
```
