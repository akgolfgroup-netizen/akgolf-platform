### Performance

| | |
|---|---|
| **Pris** | 1 600 kr/mnd |
| **Okter** | 2 x 20 min per maned |
| **Booking-vindu** | 7 dager |
| **Maks bookinger/uke** | 1 |
| **Malgruppe** | Klubbspillere som trener 1-2 ganger i uken |
