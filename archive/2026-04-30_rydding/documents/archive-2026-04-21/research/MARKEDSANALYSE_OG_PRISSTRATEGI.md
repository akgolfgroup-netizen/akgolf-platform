## 2. DIN NÅVÆRENDE PLAN: 299 kr/mnd Flat Rate

### Problemet med én flat pris:

```
PROBLEMET:
┌─────────────────────────────────────────────────────────────┐
│  Pris: 299 kr/mnd (~$30 USD)                               │
│                                                             │
│  Hva skjer:                                                 │
│  • 80% av norske golfere (HCP 20+) synes dette er for dyrt │
│  • De dropper ut før de ser verdien                        │
│  • Du får aldri kritisk masse                               │
│                                                             │
│  Konverteringsrate estimat: 2-5% av trial-brukere          │
│  (bransjestandard for SaaS: 15-25% med god freemium)       │
└─────────────────────────────────────────────────────────────┘
```

### Hva 299 kr/mnd signaliserer:
- **Positivt:** Premium produkt, seriøst, "dette er proff-verktøy"
- **Negativt:** Utenom rekkevidde for de fleste, "jeg er ikke god nok til dette"

✓ Mental tracking (forbedringsverktøy)

ELITE må inneholde:
✓ TrackMan (bevis på "jeg er seriøs")
✓ Coaching (høy verdi-persepsjon)

## 6. ÅRLIG VS MÅNEDLIG

### Psykologi bak årlig betaling:

```
PRIS-EKSEMPEL (Spiller Pro):

Månedlig:  149 kr/mnd × 12 = 1.788 kr/år
Årlig:     1.490 kr/år (sparer 298 kr = 17%)

FOR DEG:
✓ Bedre cash flow (penger opp front)
✓ Lavere churn (de har allerede betalt)
✓ ~30% vil velge årlig hvis rabatt er 15-20%

FOR BRUKER:
✓ De "låser" seg til forbedring
✓ Mindre "smerte" (én transaksjon vs 12)
✓ Føler de sparer penger
```

### Anbefaling:
- **Push årlig** med 15-20% rabatt
- Månedlig for de som er usikre (lettere å avslutte)
- 14-dagers gratis prøveperiode (krever kort, men ingen belastning)

## 9. ER DETTE VERDT DET FOR DEG?

### KOSTNAD vs INNTEKT (5-års perspektiv)

```
UTVIKLINGSKOSTNADER:

År 1: 1.5M kr (utvikler, design, infrastruktur)
År 2: 1.0M kr (forbedringer, support)
År 3: 800k kr (vedlikehold, nye features)
År 4-5: 600k kr/år

TOTAL INVESTERING: ~4.5M kr over 5 år

INNTEKTS-POTENSIALE:

År 1: 500 brukere × 1.5k kr = 750k kr
År 2: 2.000 brukere × 1.5k kr = 3M kr
År 3: 5.000 brukere × 1.5k kr = 7.5M kr
År 4: 8.000 brukere × 1.5k kr = 12M kr
År 5: 10.000 brukere × 1.5k kr = 15M kr

TOTAL INNTEKT: ~38M kr over 5 år
OVERSKUDD: ~33.5M kr

MEN: Dette forutsetter at:
✓ Produktet er virkelig bra
✓ Du finner product-market fit
✓ Konkurransen ikke kopierer deg
✓ Du klarer kundeanskaffelse
```

### Risikoer:

```
🔴 HØY RISIKO:
• DECADE kopieres av Arccos/18Birdies
• TrackMan lanserer egen app med lignende features
• Prispress fra gratis-konkurrenter

🟡 MIDDELS RISIKO:
• For liten målgruppe i Norge
• Brukerne "prøver" men ikke "fortsetter"
• Tekniske utfordringer med TrackMan-integrasjon

🟢 LAV RISIKO:
• Du har allerede academy (kundebase)
• Du har TrackMan (konkurransedyktig)
• Golf er voksende sport
```

## 10. MIN ENDELIGE ANBEFALING

### Hvis jeg var deg:

```
1. IKKE start med 299 kr flat rate
   → For dyrt for å få kritisk masse
   → Start med freemium

2. FOKUSER på B2B først (Mission Board)
   → Academy/klubber betaler mer og er lojale
   → Spillerportal blir "gateway drug"

3. LAG en knockout "DAG 1" opplevelse
   → Ny bruker må se verdi på 2 minutter
   → Hvis ikke: uninstall

4. TEST prisene først
   → Start med 99 kr/mnd Pro, 199 kr/mnd Elite
   → Øk gradvis etter hvert som du får brukere

5. BYGG community først
   → Social features er billigere enn AI
   → Men gir høy retention
```

### Alternativ modell: "Coaching-First"

```
Hva hvis du vender det på hodet?

1. Start med Mission Board (B2B)
   • Selg til 10 akademier à 1.000 kr/mnd
   • Garantert 10k kr/mnd inntekt

2. Spillerportal blir "inkludert"
   • Spillere ved akademiene får Pro gratis
   • Andre betaler 149 kr/mnd

3. Vekst via academies
   • Hver academy har 50-100 spillere
   • 10 academies = 500-1000 brukere raskt
   • De betaler indirekte via akademiet

FORTRINN:
✓ Forutsigbar inntekt fra dag 1
✓ Akademiene gjør markedsføringen for deg
✓ Du bygger økosystem, ikke bare app
```
