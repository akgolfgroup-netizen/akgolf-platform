2. Landing Page (Markedsside)

Brukerreise: Besøkende → Registrert kunde

Ankommer side Ser Hero Scroller tjenester Sjekker priser Leser testimonials Klikker "Book"

Seksjoner på Landing Page

Navigation

  • Logo (link til hjem)
  • Menylenker: Hjem, Coaching, Book, Priser
  • CTA: "Logg inn" (ghost) + "Kom i gang" (primær)
  • Mobil: Hamburger-meny

Hero Section

  • Status-badge ("Ledige timer i dag")
  • Headline (stor, bold)
  • Subheadline (beskrivelse)
  • CTA-knapper: Primær + Sekundær
  • Status-bar (system online, trenere, ledige timer)

Tjenester

  • Seksjonstittel + intro
  • 3 pakke-kort (Starter, Pro, Elite)
  • Pris, features, CTA per kort
  • 5 tjenestetyper (ikoner): Individuell, Gruppe, Simulator, On-Course, Video

Trenere

  • Grid med trener-kort
  • Foto, navn, spesialisering
  • Rating + antall anmeldelser
  • Link til full profil

Testimonials

  • Sitat med stjerner
  • Navn, bilde, handicap-forbedring
  • Carousel eller grid

FAQ

  • Accordion-format
  • 5-8 vanlige spørsmål
  • Ekspander/kollaps animasjon

CTA Section

  • Avsluttende oppfordring
  • Stor CTA-knapp
  • Sekundær link ("Kontakt oss")

Footer

  • Logo + kort beskrivelse
  • Lenkegrupper: Tjenester, Om oss, Support
  • Kontaktinfo
  • Sosiale medier
  • Copyright + personvern

3. Booking System

Komplett booking-flyt

1. Velg tjeneste 2. Velg trener 3. Velg tid 4. Dine opplysninger 5. Betaling 6. Bekreftelse

Steg-for-steg

Steg 1: Velg tjeneste

Bruker velger type coaching-time.

ElementBeskrivelseInteraksjon
TjenestetyperKort med ikon, navn, varighet, prisKlikk for å velge (radio)
Pakke-valgStarter/Pro/Elite (påvirker pris)Dropdown eller toggle
Varighet30/45/60/90 minVelg fra liste
Neste-knappDisabled til valg er gjortNavigerer til steg 2

Steg 2: Velg trener

Bruker velger foretrukket instruktør.

ElementBeskrivelseInteraksjon
Trener-kortAvatar, navn, spesialisering, ratingKlikk for å velge
"Ingen preferanse"Alternativ for første ledigeCheckbox/toggle
FilterSpesialisering (putting, driver, etc.)Dropdown-filter
Tilbake/NesteNavigasjon mellom stegKnapper

Steg 3: Velg dato & tid

Kalender og tidsluke-visning.

ElementBeskrivelseInteraksjon
Uke-navigatorPiler for forrige/neste ukeKlikk for å bytte uke
Dag-selector7 dager synlig, dato + ukedagKlikk for å velge dag
Tidsluke-gridLedige tider (grønn), opptatt (grå), valgt (svart)Klikk for å velge tid
LokasjonHvis flere steder finnesDropdown

States

  • Ledig — Kan velges
  • Få igjen — Under 3 plasser
  • Opptatt — Ikke klikkbar
  • Valgt — Brukerens valg

Steg 4: Dine opplysninger

Kontaktinfo og tilleggsinformasjon.

FeltTypeValidering
FornavnTextPåkrevd
EtternavnTextPåkrevd
E-postEmailPåkrevd, gyldig format
TelefonTelPåkrevd, norsk format
HandicapNumberValgfritt, 0-54
KommentarTextareaValgfritt, maks 500 tegn
VilkårCheckboxMå godkjennes

Eksisterende bruker: Hopp over — data hentes fra profil.

Steg 5: Betaling

Stripe Checkout integration.

ElementBeskrivelse
OppsummeringTjeneste, trener, tid, pris
RabattkodeInput + "Bruk"-knapp
BetalingsmetodeKort (Stripe Elements)
Vipps-alternativRedirect til Vipps
TotalPris inkl. MVA
"Betal"-knappDisabled under prosessering

Error states

  • Kort avvist → Vis feilmelding, prøv igjen
  • Timeout → Vis melding, tilby ny betaling
  • Duplikat → Sjekk om booking allerede eksisterer

Steg 6: Bekreftelse

Suksess-side etter betaling.

ElementBeskrivelse
Suksess-ikonCheckmark i sirkel
Overskrift"Booking bekreftet!"
Booking-detaljerTjeneste, trener, dato, tid, lokasjon
ReferansenummerUnik booking-ID
E-post-bekreftelse"Kvittering sendt til din e-post"
Kalender-knappLegg til i Google/Apple Calendar
Portal-link"Gå til din portal"

Booking-administrasjon (etter booking)

Endre booking

  • Tilgjengelig inntil 24t før
  • Velg ny dato/tid
  • Bekreft endring
  • E-post med oppdatert info

Kanseller booking

  • Over 24t: Full refusjon
  • Under 24t: 50% refusjon
  • Under 2t: Ingen refusjon
  • Bekreftelse via e-post
