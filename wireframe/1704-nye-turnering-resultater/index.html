Anbefaling: Option 1 — Scorecard Hero (med komponenter fra 3, 4 og 5 som tabs)

Scorecard Hero gir det spilleren vil se først etter en turnering: plassering, score, scorecard og hvorfor — på én skjerm. Scorecard er det universelle språket i golf og forventes å være synlig uten ekstra klikk. Runde-pills lar spilleren bytte mellom R1/R2/R3 uten å miste kontekst.

Implementasjons-flow:

  • Lever Option 1 som standard-detaljvy på /portal/turneringer/resultater/[id].
  • Listevy (Option 2) blir index /portal/turneringer/resultater — klikk kort åpner Option 1.
  • Tabs øverst i detaljvy: "Scorecard" (Opt 1) · "Runder" (Opt 3) · "Felt" (Opt 4) · "Læring" (Opt 5).
  • Lessons Learned (Opt 5) genereres retrospektivt fra TournamentPrep + Claude API når turneringen blir markert closed.
  • Field-Comparison (Opt 4) leverer histogram og cut-line fra GolfBox leaderboard-API.

Teknisk: GolfBox result-API for leaderboard, plassering, cut-line. Round-tabellen (tournament-markert) holder shot-data og SG per hull. Claude genererer AI-quote + 3 fokuspunkter via TournamentPrep (retrospektivt mode). Scorecard-ikoner (birdie/eagle/bogey) gjenbruker runde-live-komponenter.
