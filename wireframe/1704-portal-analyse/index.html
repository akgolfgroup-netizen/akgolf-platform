Anbefaling: Option 1 — Bento Grid

Gir full oversikt på én skjerm uten å overvelde. KPI-raden (GIR / FW / Putts / Scrambling) svarer "hvordan spiller jeg?" på 5 sek, HCP-trenden gir emosjonell kontekst, SG-breakdown lokaliserer svakhet, og pro-gated TrackMan-kortet skaper konvertering til Pro-tier. Samme komponenter som Statistikk — konsistens i portalen.

Strategisk flow: Implementer Option 1 som default for /portal/analyse. Option 4 (Weak Spots) kobles på som /portal/analyse/svakheter for elever med 30+ runder. Option 2 (Area Focus) blir klikk-inn-detalj fra SG-raden (Opt 1 SG-rad → Opt 2 dybde). Option 3 (Training vs Results) reserveres for Elite med 90d+ data. Option 5 (Quick Actions) er tom-state-variant for brukere med <5 runder registrert.
