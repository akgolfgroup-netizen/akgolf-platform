Anbefaling: Option 1 — Discover (med My Calendar som viktig fallback-tab)

Discover er den eneste visningen som dekker alle fire kildene (lokalt, NGF, PGA, Euro) i én enhetlig IA. Søk + filter-chips + kort-grid er et kjent mønster og lavkomplekst å bygge. Påmeldings-panelet og forberedelses-checklist ligger ved siden av listen — brukeren slipper å navigere bort for å handle.

Implementasjons-flow:

  • Lever Option 1 som primær visning på /portal/turneringer.
  • Bytt mellom "Alle" og "Mine" (Option 3) via tab-toggle i toolbar.
  • Kart (Option 2) som sekundær view-toggle — krever Mapbox, skyver til Fase 2.
  • Pro Tour (Option 4) som dedikert underseksjon når brukeren klikker PGA/Euro-chip.
  • Level-kortene (Option 5) kan gjenbrukes som entry-point på Oversikt-siden.

Teknisk: Lokalt + NGF via Golfbox-sync. PGA/Euro via DataGolf API. Plan-nivå-dekning av påmeldingsavgift kobles mot CoachingPackage i Prisma. Forberedelses-checklist genereres fra turneringens type + dager igjen.
