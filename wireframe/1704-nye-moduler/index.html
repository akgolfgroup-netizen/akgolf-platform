Anbefaling: Option 1 — Module Shop

Matcher portalens etablerte bento-stil direkte. Nåværende Pro-abonnement som svart hero (1 600 kr/mnd, +348 kr add-ons, totalt 1 948 kr, fornyes 17. mai). Kategori-grupperte modul-kort i 3-kolonne grid: Performance (TrackMan, Video, Turnering), Coaching & mental (AI Plus, Mental), Familie & foreldre (Forelder-view, Familie). Aktive moduler har tykk border og "Aktiv"-pill. Ett blikk viser: tilgjengelige moduler, priser, hva som er aktivt — og én klikk for å legge til/fjerne. Stripe Customer Portal som fallback for avanserte operasjoner.

Strategisk flow: Option 1 = default på /portal/abonnement/moduler. Option 2 (Bundle Comparison) åpnes som modal ved "Bytt bundle"-knapp. Option 3 (Active Stack) egner seg som widget på abonnement-oversikten (/portal/abonnement). Option 4 (Use-Case Driven) egen side /moduler/mal — lenke fra onboarding og sidebar. Option 5 (Price Calculator) egen side /moduler/kalkulator — for power-brukere som bygger store stacks. Alle fem mønstre er komplementære; Shop er entrypoint.
