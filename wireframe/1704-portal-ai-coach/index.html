  • 1 — Klar bruks-setting: "Perfekt på rangen" — kontekst signaliseres tidlig. Bytte til tekst alltid tilgjengelig.
  • 2 — Mic som helt: Stor sentral knapp, ingen andre elementer konkurrerer. Radiell glow viser lyttetilstand.

  • — Risiko: Voice UX er fortsatt umoden. Fallback til tekst må være friksjonsfri.

UX-notater

Anbefaling: Option 1 — Chat Classic

Trygg MVP med rask time-to-market. Claude API + eksisterende Prisma-data = funksjonell AI Coach på få uker. Chat-mønsteret er umiddelbart kjent for brukeren, kontekst-panelet til høyre gir åpenhet om datagrunnlaget (tillitsbygging). Brukeren kan spørre alt — ingen kurering av prompts kreves.

Strategisk flow: Start med Option 1. Legg til Option 2 (Quick Insight-kort) som hurtigprompt-rad øverst i chat når vi ser hvilke 5–6 spørsmål brukerne stiller mest. Option 4 (Action Driven) bygges inn i Option 1 ved å legge til action-knapper under hvert AI-svar — ingen egen side nødvendig. Option 5 (Dashboard Mode) reserveres for Pro/Elite-tier som premium-oppgradering. Option 3 (Voice) er tekniskt umoden — vent til Whisper/TTS-pris er akseptabel.
