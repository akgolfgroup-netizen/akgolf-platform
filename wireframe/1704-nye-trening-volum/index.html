Anbefaling: Option 1 — Stacked Bars

Option 1 svarer paa spoersmaalet paa 2 sekunder via KPI-raden, og gir trend over tid via 12-ukers stacked chart. Detaljlisten under lar spilleren validere mot faktiske okter. Farget sone-coding (Teknikk, Fysikk, Slag, Spill, Turnering) brukes konsekvent paa tvers av KPI-dots, stack-segments og listen — ett visuelt sprak.

Strategisk flow: Implementer Option 1 som default for N06. Vurder Option 2 (5-Ring) som egen widget paa Dashboard (N01) for motivasjon. Option 4 (Goal Tracker) kan bli en sub-view naar TrainingPlan aktivt har satt maal per omraade. Option 3 og 5 kan vaere sekundaere visninger via tab-bytte senere.
