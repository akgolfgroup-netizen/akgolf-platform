Anbefaling: Option 1 — Linear Steps

Trygg, forutsigbar, matcher portal-DNA (booking-wizard eksisterer allerede med samme mønster). Brukere som logger seg inn forventer et wizard-mønster når det står "Ny plan". Option 2 (Pyramid First) er vakker for erfarne brukere, men krever at brukeren forstår pyramiden fra start — det blir terskel.

Strategisk flow: Start med Option 1 som MVP-implementasjon. Vurder Option 2 som "Avansert modus" for Pro/Elite-tier etter 3 måneder. Option 4 (Chat Wizard) kan bygges på toppen når AI-veiledning er moden — da blir chat naturlig fremfor "bolt-on".
