Anbefaling: Option 1 — Game Hub

Game Hub er den eneste retningen som dekker alle kjernebrukerne i én skjerm: opprett sesjon, se aktive, join-med-kode, og finn bane. Challenge Mode (2) kan legges som delkomponent senere for gamification. Formatet matcher portal-arkitekturen — light canvas, sidebar, kort-mønster — og krever minst teknisk løft for go-live.
