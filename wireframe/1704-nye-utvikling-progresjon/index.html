Anbefaling: Option 1 — Five Rings

Five Rings leverer eksakt det briefen krever: fem NGF-områder synlig på 5 sekunder, med trend og neste mål per rad. Rings som visualisering er etablert mønster (Apple Activity, Whoop) og krever ingen opplæring. Kombinerer raskt overblikk (rings) med detaljert aksjon (rad-liste) i ett skjermbilde. Fungerer for alle tiers uten modus-veksling.

Strategisk flow: Option 1 som default for N01. Option 2 (Pyramid Climb) vurderes som "NGF-visning" toggle for coaches og pedagogisk onboarding. Option 3 (Data Grid) passer som underside "Alle moduler" for power-users. Option 4 (Timeline Story) brukes i månedlig e-post-rapport, ikke som hovedskjerm. Option 5 (Gamified Checkpoints) er naturlig standard for Junior Academy-tier — ikke for Pro/Elite.
